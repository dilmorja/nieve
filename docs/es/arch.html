Introducción

La Arquitectura de software es el diseño de más alto nivel de la estructura de un sistema. Consiste en un conjunto de patrones y abstracciones coherentes que proporcionan un marco definido y claro para interactuar con el código fuente del software.
La documentación describe de forma detallada cada una de las capas y de los niveles que conforman la arquitectura de este proyecto. De esto de obtienen bases sólidas y escalables que serán útiles durante toda la etapa de desarrollo.
